## 📋 What is mcp-superset?

mcp-superset is a server that helps you manage Apache Superset. It combines over 128 tools for building dashboards, viewing charts, handling datasets, running SQL queries, and controlling user access. This lets you organize your data and reports in one easy place.

Apache Superset is a popular platform used to create interactive data dashboards and visualize data without needing programming skills. mcp-superset adds extra features to make managing the server simpler and more secure.
